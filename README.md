# Full-Cycle Phishing Attack Simulation
**Offensive Security Research | MITRE ATT&CK® T1566.001**

## Project Overview
This project demonstrates a full-cycle phishing simulation conducted in a controlled lab environment. The focus is on infrastructure weaponization, credential exfiltration, and bypassing browser trust warnings.

## Tech Stack
* **OS:** Kali Linux
* **Tools:** Wget
* **Backend:** PHP
* **Security:** SSL/TLS & .htaccess

## Methodology
1. **Reconnaissance:** Domain acquisition and hosting setup.
2. **Weaponization:** Mirroring target UI using `wget`.
3. **Exfiltration:** Credential capture via custom PHP script.
4. **Trust Layer:** Enforcing HTTPS to maximize credibility.
5. **Validation:** Proof of Concept and data verification.
